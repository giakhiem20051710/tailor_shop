Phân Tích Đề Thi & Trọng Tâm Ôn Tập

Part 1: 20 Câu Trắc Nghiệm (Multiple-choice)

Thường xoay quanh những cạm bẫy của Java Core. Hãy tập trung ôn:

  • OOP: Tính Kế thừa, Đa hình (Overloading vs Overriding), Đóng gói, Trừu tượng (Abstract class vs Interface).
  • Java Core: String Pool (so sánh == và .equals()), final, static keyword.
  • Collections Framework: Điểm khác biệt giữa ArrayList, LinkedList, HashSet, HashMap.
  • Exceptions: Checked vs Unchecked Exception, cấu trúc try-catch-finally (lệnh finally luôn chạy).
  • SQL Cơ bản: SELECT, JOIN (Inner, Left), GROUP BY, HAVING, subqueries.

Part 2: 2 Bài Code Chạy Testcase (Math & Array)

Họ chỉ rõ 2 chủ đề luôn rồi, mức độ khó ngang ngửa LeetCode Easy. Bạn Code bằng Java và phải compile thành công.

Math Challenge Bài toán dùng vòng lặp, toán học: Kiểm tra số Nguyên tố (Prime), Số hoàn hảo, Dãy Fibonacci, Tính giai thừa, Đảo ngược số (Reverse Integer), Kiểm tra số đối xứng (Palindrome number).
Array Challenge Xử lý mảng (1 chiều): Tìm phần tử lớn nhất/nhỏ nhất, Đảo ngược mảng, Tìm 2 số có tổng bằng target (Two Sum), Tìm phần tử xuất hiện nhiều nhất, Xóa phần tử trùng lặp (Remove duplicates).

Part 3: EnglishScore App (40 Phút)

Đây là bài test Tiếng Anh của British Council trên điện thoại. Gồm: Ngữ pháp, Từ vựng, Nghe và Đọc hiểu. Mức độ khá dễ (Cỡ A2-B1). Nhớ tải App trước ở nhà, lập tài khoản và làm thử một bài Mock Test trên đó cho quen giao diện nhé!

Lộ Trình 3 Ngày Chạy Nước Rút

Tối Nay (T6)
Ôn Java Core & Trắc nghiệm: Giở lại mấy bài học Java Core. Đọc kỹ lại cách String hoạt động, phân biệt Interface và Abstract Class. Ôn lại cấu trúc dữ liệu Collections (List, Set, Map).
Thứ 7
Nghiền nát LeetCode vòng gửi xe: Mở LeetCode hoặc HackerRank, search các bài chuyên về Math và Array cấp độ Easy. Làm khoảng 5 bài Math và 5 bài Array bằng Java. Ghi nhớ các hàm Math.abs(), Math.max() và cách dùng vòng lặp for thần thánh.
